# pacc

propetty path utils

```js
import { getAttribute, expand } from "pacc";

const result = getAttribute({ a: [0,{ b: 4 }]}, "a[1].b");
// result === 4

const result = expand("${a + 1}",{ root: { a: 2 }});
// result === 3

```
